# Contributing to Tech Interview Cheatsheets

First off, thanks for taking the time to contribute! 🎉

The goal of this repository is to compile the most advanced, obscure, and highly technical interview questions for Senior roles in AI, Data Science, and Data Engineering. We strictly want to avoid trivial questions (e.g., "What is a Neural Network?").

## How Can I Contribute?

### 1. Suggesting New Questions
If you recently had a grueling technical interview and faced a question that stumped you because of a weird edge case, we want it!
- Open an Issue with the tag `[New Question]`.
- Provide the exact question.
- Provide a detailed, mathematically or technically sound answer.

### 2. Submitting Pull Requests
- Fork the repository.
- Create a new branch: `git checkout -b feature/add-new-question`.
- Add your question to the relevant Markdown file using the interactive format (`<details>` and `<summary>`).
- Make sure your explanation is concise but deeply technical.
- Push your branch and open a Pull Request.

## Interactive Formatting Rules
All new questions must follow our interactive flashcard structure:

```html
<details>
<summary><b>Q99: [Your complex question here?]</b></summary>
<br>

**Answer:**
Your highly detailed, technical answer goes here.
</details>
```

We review all PRs within 48 hours. Thank you for making the community smarter!
